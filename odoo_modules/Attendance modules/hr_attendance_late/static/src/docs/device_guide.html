Employee Device

Authorize one phone per employee with a Device ID and PIN, and block lost or replaced phones without losing the record. The full step-by-step manual is on the Open in PDF doc button.

1. What it does

The Employee Attendance app only accepts a check-in or check-out from a phone that HR has registered for that employee. Each employee has one device on file, identified by a 10-digit Device ID and protected by a PIN. While the device is Active the employee can mark attendance from it; turn Active off to block a lost or replaced phone without deleting its history.

2. Where to find it

Menu: Attendances → Devices (top bar, next to Late Tracking and Field Attendance). This is the only place devices are added or edited. You get a read-only list with one row per employee showing Name, Device ID, Device Name, Active, Device Added On and Last Used.

The employee's own Devices tab (on their profile) shows the same details but is view-only — registration happens only from Attendances → Devices.

3. Register a device & PIN

  1. Ask the employee for their 10-digit Device ID.
  2. Open the employee from Attendances → Devices (click their row to open the device form).
  3. Type the Device ID exactly as provided.
  4. Under PIN Authentication, set a PIN — it verifies the employee when marking attendance.
  5. Make sure Active is switched on, then save.
  6. Optionally set the Device Type (Android or iOS). Device Name and Last Used fill in automatically once the device is used — you don't type them.

4. Managing devices

  • Block a lost or old phone: turn Active off instead of deleting — the record and its history stay, and you can switch it back on later.
  • Replace a device: each employee can have only one device. To move to a new phone, update the existing Device ID (or deactivate it first) — adding a second device is blocked.
  • Check recent activity: use the Last Used column/field to confirm a phone is active and marking attendance.

5. Reference: Device fields

FieldWhat it means
Device IDThe 10-digit code that identifies the employee's authorized device.
Device NameThe device's name. Fills in automatically once the device is in use.
Device TypeAndroid or iOS. Defaults to Android.
ActiveWhether the device may mark attendance. Turn off to block without deleting.
PINThe code that verifies the employee when marking attendance.
Device Added OnWhen the device record was created.
Last UsedWhen the device last marked attendance. Fills in automatically.
Remember: only one device per employee — deactivate or update the existing one before adding a new phone. Attendance can be marked only from a registered AND Active device, verified by PIN. Lost a phone? Turn Active off rather than deleting so you keep the history and can re-enable it later.
