Employee Monthly Report

Working days, present days, late/leave deductions and the final salary — per employee, per month.

1. Generating

Menu: Employee Report. Pick the month/year, company and employees, click Generate, then Refresh any time to recompute from the latest config and attendances.

2. Days

  • Working Days — checked weekdays + half-day Fridays (0.5 each), minus holidays; recalculated per month (e.g. June = 27, July = 27.5).
  • Present Days — days actually attended (a half-day-Friday attendance counts 0.5).
  • A working day with no attendance and no leave shows as Not Entered; future days are not shown.

3. Salary — earnings based

You earn one day's pay for each day worked; absent / unpaid days simply aren't earned.

daily rate = wage / working days in month earned = (present days + paid-leave days) × daily rate Final = max(0, earned − late deductions) Total Deduction = Wage − Final
So 0 present → Final 0, and full attendance → full wage (minus any late). The daily rate is rounded to currency precision so amounts match the leave requests.

List columns are selectable from the list's options (sliders) menu; deduction cells are red, Final is bold/green.

5. Day-wise details & export

  • The Day-wise Details tab lists each day (Present / Late / Leave / Not Entered / Holiday) with the office-time Check In and Check Out.
  • Export to PDF and Excel — both match the on-screen figures.
