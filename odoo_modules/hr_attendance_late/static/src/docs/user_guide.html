1. Overview

Employees check in (kiosk, mobile or manual). Odoo stores every check-in in UTC. The system converts it to the Office Timezone, decides if it is late against the configured start time, counts it against a monthly grace, applies a deduction (slab or hourly), lets the employee add a reason and request a waiver, and rolls everything up into the monthly report and final salary.

Check in → convert to Office Timezone → which session / expected start → late? (threshold) → which late # this month → past grace? → slab/hourly deduction → reason → waiver? → month-end report → final salary.

3. Working Days & Half-Day Fridays

Working days for a month = the checked weekdays (minus public holidays). A half-day Friday (a Friday in a configured position, e.g. 2,4,5 = 2nd/4th/5th Friday) counts as 0.5 — even if Friday itself is unchecked.

Example — June (Friday off, positions 2,4,5): 26 full days + 0.5 (2nd Fri) + 0.5 (4th Fri) = 27 working days. July (5 Fridays) → 26 + 1.5 = 27.5. It recalculates per month automatically.

4. How late detection works

  • The UTC check-in is converted to the Office Timezone.
  • The session is determined; the expected start is the session's start (or the half-day-Friday start).
  • If the local check-in is later than start + Late Threshold, the record is marked Is Late with the Late Minutes measured from the start time.
Example (Office TZ Asia/Muscat): a check-in stored 04:10 UTC = 08:10 Muscat; with start 08:00 → 10 min late.

5. Deductions — Slab vs Hourly

Set by Deduction Mode. Both apply only after grace and are 0 if waived.

Fixed (Slab-based)

Menu: Late Tracking → Deduction Slabs. The late minutes choose which slab applies and that slab's flat amount is deducted (not per-minute). A slab matches when From ≤ late_minutes ≤ To (To = 0 means unlimited).

Hourly Wage

Scales with how late you are, from the employee's salary:

hourly_rate = monthly wage ÷ working days in month ÷ daily work hours deduction = hourly_rate × ceil(late_minutes ÷ 60) (every started hour = 1 hour)
No-wage fallback (Hourly mode): if an employee has no wage set, the hourly amount is 0, so the system falls back to the matching slab amount. With no slab, the deduction is simply 0. Keep a basic slab as a safety net for unwaged employees.

6. Grace & Late-Waiver flow

  • Grace: the first N late occurrences per month (per session) are free (deduction 0); a reason is still requested.
  • Enter Late Reason: on a late attendance, the employee clicks Enter Late Reason and types why.
  • Waiver: Menu Late Tracking → Late Waiver Requests. The employee picks the late attendance, gives a reason, and submits. A manager Approves (sets the record to Waived, deduction → 0) or Rejects.

7. Leave Policy

Menu: Leave Requests + Leave Policy Configuration.

  • A leave is paid only when an enabled paid-leave policy with remaining quota grants it; otherwise (no policy, or paid leave off) it is unpaid.
  • Unpaid-leave deduction = wage ÷ working days in month × unpaid days (when unpaid deduction is enabled). The daily rate is rounded so amounts match the report.

8. Employee Monthly Report

Menu: Employee Report. Pick month/year and employees, Generate, then Refresh.

  • Working Days (half-day-Friday aware) and Present Days (half-day weighted).
  • Final Amount is earnings-based: you earn one day's pay (wage ÷ working days) for each day worked (present days + paid leave); late arrivals reduce it; absent / unpaid days simply aren't earned. So 0 present → 0 final, full attendance → full wage.
  • Leave Deduction shown separately; Total Deduction = Wage − Final.
  • Days with no attendance and no leave show as Not Entered; future days are not shown.
  • Export to PDF and Excel; columns are selectable from the list's options menu.

9. Office-time display

Standard Odoo date columns always show the logged-in user's timezone. To show office time everywhere, the attendance lists/kanban/form, the Late Waiver, the Employee Report (day-wise + PDF + Excel), the Field Attendance list and the Late Attendance PDF all show dedicated Office-Time values (and the built-in columns on lists are hidden). The late logic and the Office-Time display are always anchored to the Office Timezone, regardless of the viewer.

Tip: use the Print / Save as PDF button at the top to keep an offline copy. To add more documents, drop PDFs/HTML into hr_attendance_late/static/src/docs/ and link them the same way.
